# Calculator Program in C

A simple calculator program written in C that performs basic arithmetic operations using the `switch` statement.

## Features

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Handles division by zero
* Displays an error message for invalid operators

## Technologies Used

* C Programming Language
* Standard Input/Output Library (`stdio.h`)

## How to Run

1. Save the code as `calculator.c`.
2. Compile the program:

   ```bash
   gcc calculator.c -o calculator
   ```
3. Run the program:

   ```bash
   ./calculator
   ```

## Sample Output

```
Enter first number: 10
Enter an operator (+, -, *, /): *
Enter second number: 5
Result = 50.00
```
